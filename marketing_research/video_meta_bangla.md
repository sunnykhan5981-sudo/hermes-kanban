## ২. Subtitles / Captions Patterns

### Pattern A: Reel Hook (0-3s)
**Visual**: Face + text overlay
**Audio**: "দেখো!"
**Subtitle**: 
```
দেখো!
```
*(bold, yellow, center screen)*

---

### Pattern B: Tutorial Steps
**Visual**: Screen recording + zoom
**Audio**: "এখানে নাও..."
**Subtitle**:
```
ধাপ ১: ডাউনলোড করো
ধাপ ২: রেজিস্ট্রেশন
ধাপ ৩: ম্যাচে যোগ দাও
```
*(bottom third, white text with black outline)*

---

### Pattern C: Social Proof
**Visual**: Balance screenshot → zoom
**Audio**: "৫,০০০ এভার!"
**Subtitle**:
```
গতকাল ৫০০
আজ ৫০০০
```
*(large numbers, animated count-up)*

---

### Pattern D: Challenge / CTA
**Visual**: Host pointing at camera
**Audio**: "চ্যালেঞ্জ ট্যাগ কর!"
**Subtitle**:
```
তুমি চ্যালেঞ্জ কর?
ক্যামেন্টে ↓
```
*(bottom, arrow pointing to comment box)*

---

### Pattern E: Live Event
**Visual**: LIVE badge + countdown
**Audio**: "আজ রাতে ৮টা"
**Subtitle**:
```
🔴 লাইভ
আজ রাতে ৮টা
পুরস্কার ১০,০০০
```
*(top-left LIVE, center text)*

## ৬. Subtitle Writing Rules

1. **Timing**: ২-৫ সেকেন্ড-per subtitle line
2. **Length**: ২-৩ शব्द per line (readable at 9x speed)
3. **Font**: Noto Sans Bengali, bold, white with black outline
4. **Position**: Center or lower third (avoid top — FB UI covers it)
5. **Emphasis**: Use size/color for key words (numbers, CTA)
6. **Language**: Pure Bangla, no English/Hindi
7. **Sync**: Match subtitle timing with speech pauses
