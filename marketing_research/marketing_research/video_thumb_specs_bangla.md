# LUDOLAND Facebook Video Thumbnail Strategy

## Thumbnail Principles (Bangladesh Market)

### Core Rules
- **Text**: Bangla only, max 3-4 words, bold font
- **Face**: Real Bangladeshi face (18-35) expressing emotion
- **Color**: High contrast — orange/yellow/red backgrounds
- **Logo**: LUDOLAND watermark bottom-right
- **Size**: 1280x720 (16:9) for Feed, 1080x1920 (9:16) for Reels

---

## Thumbnail Templates by Category

### 1. Curiosity Hooks
**Template**: Face + question text + arrow pointing up
- Face: Wide eyes, mouth slightly open
- Text: "৩ মিনিটে ২০০ স্কোর?"
- Background: Gradient orange to red
- Arrow: Yellow pointing to "দেখো" button

### 2. Social Proof
**Template**: Screenshot + badge + face reaction
- Screenshot: Balance screen showing ৫০০০+ number
- Badge: "ভাইরাল" star icon top-left
- Face: Celebrity expression (joy/surprise)
- Background: Green gradient (trust)

### 3. Tutorial/How-To
**Template**: Step number + before/after + tool icon
- Text: "ধাপ ১/৩" (Step 1/3)
- Icon: Game controller or dice
- Before: Ludo board empty
- After: 4 tokens home
- Background: Blue gradient (learning)

### 4. Challenge/Competition
**Template**: VS badge + two faces + timer
- Left: Your face (confident)
- Right: Opponent face (nervous)
- Center: "VS" badge
- Timer: "২৪:০০:০০" countdown
- Background: Red vs blue split

### 5. Live Event
**Template**: LIVE badge + face + clock
- Badge: Red "LIVE" top-left
- Clock: "আজ রাতে ৮টা"
- Face: Host with mic
- Background: Dark with spotlight effect

### 6. Festival/Offer
**Template**: Festival icon + offer text + decorative border
- Icon: Date palm (Eid), mango (Puja), flag (26 March)
- Text: "৫০% বোনাস!"
- Border: Golden diya/marigold pattern
- Background: Festive color (green/red/yellow)

---

## Thumbnail Checklist (Pre-upload)

- [ ] Text is Bangla, no Hindi/English mixed
- [ ] Face is Bangladeshi, age 18-35
- [ ] Emotion matches video tone
- [ ] Logo watermark present
- [ ] No forbidden words visible (Cash/Withdraw/Win/Bet/Gambling/Deposit/Prize)
- [ ] Safe words used: Add Balance / Redeem / Board Fee / Service Fee
- [ ] High contrast, readable on mobile
- [ ] No misleading claims
