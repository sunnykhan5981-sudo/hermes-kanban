## Description Formula (Quick Reference)

```
[EMOJI] HOOK (1 line, Bangla)
[Story/Proof] (1-2 lines)
[CTA] (Join/Share/Download)
[Hashtags] (10-15)
[Link] (Optional)
```

**Length**: 100-300 characters (Facebook optimal)
**First 125 chars**: Most visible in feed
**Hashtags**: Place at end or first comment

## Description Audit Checklist

- [ ] First line hook is Bangla, no English
- [ ] No forbidden words: Cash/Withdraw/Win/Bet/Gambling/Deposit/Prize
- [ ] Safe words used: Add Balance/Redeem/Board Fee/Service Fee
- [ ] Hashtags are relevant, rotated
- [ ] CTA is clear and actionable
- [ ] Link is correct and shortened
- [ ] Emoji usage ≤ 3 emojis
- [ ] Length ≤ 300 characters
